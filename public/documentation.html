Documentation Page for Jackson's movie API

INSTRUCTIONS

This app contains info on movies!

Here you'll find information on movies, directors, genres

Request URL HTTP Response
Return a list of ALL movies to the user /movies GET json object of all movies
Return data (description, genre, director, image URL, whether its featured or not) about a single movie by title to the user /movies/title GET json object of the selected movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genreName GET json object of genre
Return data about a director (bio, birth year, death year) by name /movies/directorName GET json object of director
Allow new users to register /users POST message reporting successful registration
Allow users to update their user info (username) /users/username POST message saying info was updated
Allow users to add a movie to their list of favorites /users/username/movies/MovieID POST message saying movie was added to favorites
Allow users to remove a movie from their list of favorites /users/username/movies/MovieID DELETE message confirming deletion of movie from favorites
Allow existing users to deregister /users/username DELETE message confirming deregistration
